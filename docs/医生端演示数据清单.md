# 医生端演示数据清单

## 1. 时间段与医生关系

当前后端模型不是“全院同一时刻只能有一个医生出诊”，而是：

- 一条 `SchedulingActual` 只属于一个医生。
- 一条 `SchedulingTimeSlot` 只属于一条 `SchedulingActual`。
- 因此一个具体号源时间段只会属于一个医生，不会同时绑定多个医生。
- 但不同医生可以各自拥有自己的 `08:00-08:10`、`08:10-08:20` 这类时间段。
