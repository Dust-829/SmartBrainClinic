# 医生端演示数据清单

更新时间：2026-07-10。

## 1. 时间段与医生关系

当前后端模型不是“全院同一时刻只能有一个医生出诊”，而是：

- 一条 `SchedulingActual` 只属于一个医生。
- 一条 `SchedulingTimeSlot` 只属于一条 `SchedulingActual`。
- 因此一个具体号源时间段只会绑定一个医生，不会同时绑定多个医生。
- 但不同医生可以各自拥有自己的 `08:00-08:10`、`08:10-08:20` 这类时间段。

## 12. 演示日期与图表验收边界

当前脚本中的排班和挂号样本固定为 `2026-07-07` 与 `2026-07-08`。医生工作台只统计“当日”队列，因此系统日期晚于该范围时，工作台会正确显示空队列，不能据此判断状态环图或分时段条形图失效。

要验收非空图表场景，需在演示当天生成对应日期的排班和挂号数据，或在受控演示环境中使用与 seed 日期一致的业务日期。不得为了截图直接在前端伪造人数。
